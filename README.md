# Votatis: 선거 무결성 아카이브 프로젝트



**"민주주의의 꽃 선거 논란, 기술과 팩트로 밝힌다"**


현재 대한민국의 민주주의는 잔인하게 짓밟히고 훼손되었습니다. 잃어버린 민주주의를 되찾기 위해서라도 시민들의 힘이 필요합니다.
Votatis는 시민의 힘으로 직접 선거 의혹 증거물을 수집, 아카이빙하고 검증하는 **시민 주도형 오픈소스 시빅테크(Civic-Tech) 플랫폼**입니다.

## 📌 Project Overview (프로젝트 개요)



6월 3일 치러진 제9회 전국동시지방선거 직후, 전국 곳곳에서 개표 지연 사태와 투표용지 관련 논란이 발생하고 있습니다. 현장에서 시민들이 발견한 훼손된 봉인지와 비정상적인 투표지들은 단순한 음모론으로 치부할 수 없는 명백한 '물증'이자 '기록'입니다.



문제는 이러한 의혹들이 온라인 커뮤니티상에서 산발적으로 공유되다가 시간이 지나면 휘발된다는 점입니다.



**Votatis**는 특정 단체나 개인의 소유가 아닌, 선거의 투명성을 바라는 모든 시민과 개발자들이 함께 구축하는 공익 프로젝트입니다. 분산된 증거들을 체계적인 데이터베이스로 모으고 검증 프로세스를 거쳐 영구히 기록합니다.



---



## 🛠️ Core Features (주요 기능)



### 1. 전산 데이터 및 의혹 맵핑 (Geospatial Mapping)

* 전국 투표소 및 개표소 지도 API를 기반으로, 의혹 및 물증이 발생한 위치를 실시간 시각화 맵으로 제공합니다.



### 2. 이미지 포렌식 검증 시스템 (Image Forensic Pipeline)

* 제보된 현장 사진·영상의 메타데이터(EXIF) 검증, 위변조 여부(AI 분석 및 합성 식별)를 파악하기 위한 파이프라인을 구축합니다.



### 3. 유형별 증거 아카이빙 (Structured Data Archiving)

* 투표함 수치 에러, 투표지 훼손 등 수집된 제보를 정형화된 카테고리별 데이터베이스로 분류하여 누구나 투명하게 검증할 수 있도록 공개합니다.

## 💻 Tech Stack (기술 스택 예시)

> *현재 프로젝트 초기 단계로, 아키텍처 및 스택 제안을 환영합니다.*



* **Frontend:** React.js / Next.js / Tailwind CSS

* **Backend:** Node.js (Express) / Python (FastAPI)

* **Database:** PostgreSQL / MongoDB

* **Data Visualization:** Leaflet / Mapbox / D3.js

* **Image Processing:** OpenCV / Python EXIF Tools

## 🤝 How to Contribute (기여 방법)



Votatis는 코드 한 줄부터 UI 디자인, 단순 데이터 수집까지 모든 형태의 기여(Contribution)를 환영합니다.



### 1. 개발자 (Developers)

* 리포지토리의 `Issue` 탭에서 작업하고 싶은 태스크를 확인하거나 새로 제안해 주세요.

* `Fork` 후 `Pull Request(PR)`를 통해 코드를 기여하실 수 있습니다.



### 2. 디자이너 (Designers)

* 서비스 UI/UX, 와이어프레임 디자인 및 브랜드 에셋 작업에 참여하실 수 있습니다.



### 3. 데이터 수집 및 검증자 (Data Workers & Researchers)

* 9회 지선 관련 가짜뉴스와 실제 물증을 교차 검증하고, 데이터를 정제하여 팩트체크 크루로 활동합니다.



### 4. 제보 및 문서화 (Writers)

* 현장 물증 제보 및 프로젝트 Wiki, README 번역 등 문서화 작업에 기여합니다.
